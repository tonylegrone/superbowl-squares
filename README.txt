INSTRUCTIONS:
1. Fill in the home and away team names.

2. Add all your player names.

3. Select a player and click the squares they've selected. Repeat for
   each player. After all the squares have been selected, the board will
   generate the row and column numbers randomly from 0 to 9.

4. Fill in the scores from each quarter to match up the winning square.

NOTES:
Use the "game reset" button to erase all data from the board (including
player names). This action CANNOT be undone. So, be careful.
